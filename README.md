# 🌿 Crop Doctor

Crop Doctor is an AI-powered web application that helps farmers and gardeners identify plant diseases by simply uploading a leaf image.

## 🚀 Features
- 📷 Upload plant leaf images
- 🔍 Detect diseases using AI API (Plant.id)
- 💊 Get possible cures and treatment suggestions
- 🌱 Simple and user-friendly interface

## 🛠️ Tech Stack
- Backend: Flask (Python)
- Frontend: HTML, CSS, JavaScript
- API: Plant.id API
- Image Handling: Local upload system
